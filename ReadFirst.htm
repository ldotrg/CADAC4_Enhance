Welcome to the World of Aero-Space Simulations, October 2013, Version 4.1

  • Program CADAC (Computer Aided Design of Aerospace Concepts) provides an environment, written in FORTRAN and C++ for the development of general purpose, digital computer simulations of time phased, dynamic systems. It manages input and output, generates stochastic noise sources, controls state variable integration and provides post-processing data analysis and display. CADAC has proven its adaptability to missiles, aircraft and hypersonic vehicles. The CADAC environment is suitable for 3DOF, 5DOF, and 6DOF simulations. It supports deterministic and Monte Carlo runs, as well as launch envelope and footprint generation.
  • CADAC is compatible with Windows based PCs (32 or 64 bit)
  • The FORTRAN source code is compatible with Visual FORTRAN 6.1 compilers
  • The C++ source code is compatible with MS Visual C++ compilers (free Visual C++ Express downloads)
  • CONTENT:
    • CADAC_FTN programs support the examples in the book: “Modeling and Simulation of Aerospace Vehicle Dynamics,” AIAA Education Series
    • CADAC++ programs are counterparts to the FORTRAN simulations  
    • CADAC Studio contains plotting and post processing utilities for CADAC_FTN and CADAC++
  • CADAC supports also the three Self-Study Courses:
    • "Building Aerospace Simulations in C++"
    • "Fundamentals of 6 DoF Aerospace Simulation and Analysis in C++"
    • "Advanced 6 DoF Aerospace Simulation and Analysis in C++"

INSTALLATION OF CADAC STUDIO

  • Remove all previous KPLOT, SWEEP, SWEEP++ with the Add/Remove tool
  • Click KPLOT, SWEEP, SWEEP++ for individual installation
    • KPLOT is a standalone program with options for 2DIM, 3DIM and Globe plotting, as well as stochastic data processing
    • SWEEP generates for CADAC_FTN simulations launch envelopes and footprints, including plots
    • SWEEP++ is the pre-processor for CADAC++ simulations in preparation for plotting with SWEEP

FORTRAN INSTALLATIONS

 
    \...\STUDIO\CADX          The pre-processing CADAC utilities are contained here.  These
                              are FORTRAN programs used to define the necessary input for CADAC.
                              The programs are:
                              AHEAD3          EQMAP3
                              CADIN3          FRLOC3
                              DFHEAD3         INPUT3
                              DFMOD3          MKHEAD3
 
                              Also included are the common CADAC FORTRAN files:
                              CADX3 - Executive routine
                              UTL3  - Matrix utilities
 
                              INSTALLATION - These files are installed by copying the files to the
                              hard drive and adding the executable (*.EXE) to the
                              FORTRAN compiler TOOL bar.  If the executables cannot be
                              run, the FORTRAN files should be compiled and linked to
                              produce a valid *.EXE file.

C++ INSTALLATIONS                                           

                             Build a project in the MS C++ compiler Editor using the name of the simulation.
                             Copy all source files into the project directory, load them into the project, and compile.
                             Read ‘readme.asc’ and study ‘documentation.asc’ files. 
                             Make a run with the default ‘input.asc’ file.

QUICK START FOR CADAC_FTN

After having transferred all CADAC files to your local disk, go to Directory \CADAC\STUDIO\CADX\TEST and double click on SSTO.EXE. You are running the Single Stage to Orbit test case documented in CADAC\STUDIO\DOC\QUICK31.DOC. Install KPLOT and use the TRAJ.BIN file, you just generated, to plot the trajectory over the globe using the GLOBE option (LONGITUDE=THETA, LATITUDE= PHI, ALTITUDE=HBE, LAT/LON UNITS = RADIANS ). Afterwards, read CADAC\STUDIO\DOC\QUICK31.DOC documentation and build the SSTO executable again using your FORTRAN compiler, following the instructions.
